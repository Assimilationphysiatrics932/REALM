## 📖 About the software

REALM gives you a framework for embodied reactive listening. The software uses a coarse-to-fine generative approach to process sounds. This system mimics how humans react to audio input in real-time. It understands the broad structure of incoming signals first. Then, it refines those signals into specific responses. This ensures that the output feels natural, fluid, and timely. 

You can use this tool to bridge the gap between digital signals and physical interaction. It helps your computer react to its environment. Whether you study sound, research human interaction, or build smart devices, this tool provides the foundation you need.

## 💻 System requirements

Your Windows computer requires specific hardware to run REALM smoothy. Check your system specs before you start.

- Operating System: Windows 10 or Windows 11 (64-bit).
- Processor: Intel Core i5 or AMD Ryzen 5 (2.5 GHz or faster).
- Memory: 8 GB RAM minimum (16 GB recommended).
- Graphics: Dedicated GPU with at least 4 GB of VRAM.
- Storage: 2 GB of free space on an SSD for fast data access.
- Audio: A reliable microphone and high-quality speaker setup.

Ensure your Windows drivers for audio and graphics are up to date. You can check these through the Windows Device Manager.

## 🚀 Running your first session

Once you install the software, you can start the application.

1. Double-click the shortcut icon on your desktop.
2. If the application requests permission to access your microphone, click "Allow." The tool needs your audio input to perform its listening tasks.
3. Wait for the main window to open. The initialization process takes about ten seconds as the system scans your hardware.
4. Select your primary input device from the dropdown list in the top left corner.
5. Click the "Connect" button. The light will turn green to indicate the software is active and ready to listen.

## 🛠️ Calibration for better results

Calibration ensures the system responds to your audio environment correctly.

1. Go to the "Settings" tab in the main window.
2. Locate the "Threshold" slider.
3. Speak at your normal volume while the software is active.
4. Adjust the slider until the indicator light matches your intensity. This prevents the system from reacting to background noise or quiet hums.
5. Save your settings. The program will remember these changes for your next session.

## ❓ Troubleshooting common issues

If you encounter problems, check these items first.

- The application keeps crashing: Close all other programs using the microphone. Sometimes multiple apps try to access the audio hardware at the same time.
- No response when I speak: Check the "Input" setting in the main menu. Ensure you selected the correct microphone from the list.
- The audio quality sounds poor: Check your cable connections or internet speeds if the software connects to a remote sensor.
- The window is blank: Restart the application. If the problem persists, uninstall and reinstall the program using the same steps from the download section.

Keep your machine plugged into power while you run the application. High-performance generative tasks drain battery life quickly on laptop computers.

## 💡 Tips for use

Use the software in a quiet room for the best experience. The reactive listening engine works most efficiently when it can isolate your vocal input from echo or hum.

- Keep your microphone at least six inches from your mouth.
- Avoid using speakers that output sound directly into your microphone grid. 
- You can turn off the "Visualization" mode to save on system resources if your computer runs slowly.
- Use the "Log" tab if you want to see how the software interprets your sound inputs over time. This helps you understand the generative responses better.
